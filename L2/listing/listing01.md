Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
```
77, 78, 79

b является sliceом и хранит элементы c индексами от 1(включительно) до 4(не включительно)

```
